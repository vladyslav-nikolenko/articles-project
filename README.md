# Articles App

A React application for browsing, searching, filtering, and sorting articles. Built with React Router v7 (SSR), MUI, and TypeScript.

## Tech Stack

- [React Router v7](https://reactrouter.com/) — SSR, routing
- [MUI v7](https://mui.com/) — UI components
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)

## Getting Started

```bash
npm install
```

### Development

```bash
npm run dev
```

App runs at `http://localhost:5173`.

### Production

```bash
npm run build
npm start
```

### Typecheck

```bash
npm run typecheck
```
